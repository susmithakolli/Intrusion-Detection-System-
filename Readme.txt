Readme:

Set up: Download and install Anaconda3 with python3

Python packages requirements: flask, jupyter

How to run:

Start Anaconda prompt

Right click on the prompt to start as an administrator

cd Intrusion_Detection

python intrusion_app.py

On the browser, start the web interface:

http://127.0.0.1:5000/

Running the Program:

Click on the ‘Choose File’ button and select the normal file or attack file to run

Click ‘Submit’ button

Click 'Result' button
